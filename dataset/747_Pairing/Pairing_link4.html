CryptoBib

CryptoBib is a BibTeX database containing papers related to Cryptography, with manually checked entries and uniform BibTeX data. CryptoBib can be used as a classical BibTeX database. BibTeX keys are short and easy to remember.

URL field New

2017-04-06: URLs are now indicated in the url field instead of in the field note. If you were using splncs.bst or biblatex / biber, you should not need to change anything. But if you are still using old bibtex styles, such as alpha.bst or plain.bst, you may want to move to alphaurl.bst or plainurl.bst, otherwise URLs will not be displayed anymore.

GitHub

The code is now open source, and you can now add cryptobib files as an SVN external project or a Git submodule to always stay up to date:

  • Add the submodule to your paper's project:
    git submodule add https://github.com/cryptobib/export cryptobib
  • Update cryptobib:
    ( cd cryptobib && git pull origin master )
  • Retrieve cryptobib after git clone or git pull:
    git submodule update --init
